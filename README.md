# CP Construction NWA Website

This is a static version of the CP Construction NWA website with no v0 branding.

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload these files to the repository.
3. In Vercel, click **Add New → Project**.
4. Import the GitHub repository.
5. Deploy.
6. In the new Vercel project, add:
   - `www.cpconstructionnwa.com`
   - `cpconstructionnwa.com`

Your GoDaddy DNS should already point to Vercel:
- A `@` → `76.76.21.21`
- CNAME `www` → `cname.vercel-dns.com`
